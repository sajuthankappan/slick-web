# slick-web

Slick Web UI using Svelte.js

# Technologies Used

- Svelte.js
- Sapper.js
- Bulma CSS

## How to use

Download the code [or clone the repo](https://gitlab.com/sajuthankappan/slick-web/)

create/copy .env file, at the root of the project

Install npm packages and run:

```sh
npm install
npm run dev
```
